# @tabnas/bnf

BNF / ABNF grammar compiler for the
[tabnas](https://github.com/rjrodger/tabnas) parser — takes BNF/ABNF
source and emits a tabnas `GrammarSpec`. Also ships a `tabnas-bnf` CLI.

This repository contains:

| Path | Description |
|---|---|
| [`ts/`](ts/) | TypeScript / JavaScript implementation (`@tabnas/bnf`), plus the `tabnas-bnf` CLI. |

See [`ts/README.md`](ts/README.md) for usage.
