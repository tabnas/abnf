# @tabnas/abnf

ABNF grammar compiler for the
[tabnas](https://github.com/rjrodger/tabnas) parser — takes ABNF
source and emits a tabnas `GrammarSpec`. Also ships a `tabnas-abnf` CLI.

This repository contains:

| Path | Description |
|---|---|
| [`ts/`](ts/) | TypeScript / JavaScript implementation (`@tabnas/abnf`), plus the `tabnas-abnf` CLI. |

See [`ts/README.md`](ts/README.md) for usage.
